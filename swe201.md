  - resizable array.
    - dynamic: under-the-hood, add() might call resize()
    - duplicates allowed
    - order: based on when they were inserted
  - add()
  - resize()
  - ex.1 
    ```java
      ArrayList<String> fruits = new ArrayList<>(); // Creating an ArrayList
      fruits.add("Apple");      // Adding elements
      fruits.add("Banana");
      fruits.add(1, "Orange");  // Adding element at a specific index
      fruits.get(0);            // Accessing using an index
      fruits.remove(0);         // Removing element using index
      fruits.remove("Banana");  // Removing element using value
      fruits.set(0, "Mango");   // Updating element
      System.out.println("After updating value: " + fruits);

      fruits.size();                // Size of ArrayList
      fruits.isEmpty();             // Checks whether empty
      fruits.contains("Apple");     // Checks if element exists
      fruits.clear();               // Clear the ArrayList
      fruits.indexOf("Apple");      // Index of the first occurence
      fruits.lastIndexOf("Apple");  // Last occurrence
    ```
